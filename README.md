# 다정한 밴드

MP3 파일을 업로드하면 iTunes에서 메타데이터를 자동으로 가져와 스트리밍할 수 있는 음악 플레이어 웹앱.

---

## 사용 AWS 리소스

| 리소스 | 용도 |
|--------|------|
| S3 | 프론트엔드 정적 웹 호스팅, mp3 파일 및 커버 이미지 저장 |
| RDS (MySQL) | 트랙 메타데이터 저장 |
| SNS | S3 업로드 이벤트를 Lambda로 팬아웃 |
| Lambda `fetchMetadata` | iTunes API로 앨범명·커버 이미지 조회 후 DB 업데이트 |
| Lambda `extractDuration` | mp3에서 재생 시간 추출 후 DB 업데이트 |
| EC2 | Express 백엔드 서버 호스팅 |

---

## 실행 방법

브라우저에서 아래 주소로 접속합니다.

```
http://kmucloud-06-s3.s3-website-us-east-1.amazonaws.com/
```

> 별도 설치나 로그인 없이 바로 사용 가능합니다.
> 프론트엔드는 S3 정적 웹 호스팅, 백엔드 API는 EC2(`13.222.13.232:3000`)에서 실행됩니다.

### 트랙 업로드

1. 우측 상단 곡 추가 버튼 클릭
2. 제목, 아티스트, 녹음일, 멤버, mp3 파일 입력 후 업로드
3. 업로드 완료 후 Lambda가 자동으로 앨범명, 커버 이미지, 재생 시간을 채워줍니다 (수 초 소요)

---

## 테스트 정보

샘플 트랙이 DB에 미리 등록되어 있어 접속 즉시 재생 테스트가 가능합니다.

업로드 테스트용 샘플 mp3는 아래 조건이면 정상 동작합니다.
- 형식: mp3
- 제목·아티스트: iTunes에 등록된 곡 입력 시 커버 이미지 자동 적용
