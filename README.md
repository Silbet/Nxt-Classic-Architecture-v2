# 밴드 녹음곡 플레이어

MP3 파일과 곡 이름, 아티스트 이름을 업로드하면 iTunes에서 메타데이터를 자동으로 가져와 스트리밍할 수 있는 음악 플레이어 웹앱.

---

## 사용 AWS 리소스

| 리소스 | 용도 |
|--------|------|
| S3 | 프론트엔드 정적 웹 호스팅, mp3 파일 및 커버 이미지 저장 |
| EC2 | Express 백엔드 서버 호스팅 |
| RDS (MySQL) | 트랙 메타데이터 저장 |
| SNS | S3 업로드 이벤트를 Lambda로 팬아웃 |
| Lambda `fetchMetadata` | iTunes API로 앨범명·커버 이미지 조회 후 DB 업데이트 |
| Lambda `extractDuration` | mp3에서 재생 시간 추출 후 DB 업데이트 |


---

## 실행 방법

브라우저에서 아래 주소로 접속합니다.

```
http://kmucloud-06-s3.s3-website-us-east-1.amazonaws.com/
```

> 로그인 없이 바로 사용 가능합니다.

### 트랙 재생

1. 곡 목록에서 원하는 트랙의 커버 이미지 또는 제목을 클릭하면 재생됩니다.
2. 상단 플레이어에서 재생/일시정지, 이전/다음 곡, 탐색이 가능합니다.
3. 트랙 우측 화살표를 클릭하면 멤버 정보를 확인할 수 있습니다.
4. 우측 상단 최신순/오래된순 버튼으로 정렬 순서를 변경할 수 있습니다.

### 트랙 업로드

1. 우측 상단 **곡 추가** 버튼 클릭
2. 제목, 아티스트, 녹음일, 멤버, mp3 파일 입력 후 추가 버튼 클릭
3. 업로드 즉시 목록에 추가되며, 수 초 후 Lambda가 자동으로 앨범명·커버 이미지·재생 시간을 채워줍니다.

### 트랙 삭제

1. 우측 상단 **곡 삭제** 버튼 클릭 (편집 모드 진입)
2. 삭제할 트랙 우측에 나타나는 휴지통 아이콘 클릭
3. 확인 다이얼로그에서 확인을 누르면 DB 및 S3 파일이 함께 삭제됩니다.
4. **곡 삭제** 버튼을 다시 클릭하면 편집 모드가 해제됩니다.

## 테스트 정보

`samples/` 폴더에 테스트용 샘플 mp3가 포함되어 있습니다.

| 항목 | 값 |
|------|-----|
| 제목 | 청춘만화 |
| 아티스트 | 이무진 |
| 녹음일 | {오늘날짜} |
| 멤버 | {본인이름} |
| 파일 | [청춘만화.mp3 다운로드](samples/청춘만화.mp3) |

업로드 후 수 초 후에 새로고침 시 Lambda가 iTunes에서 앨범명과 커버 이미지를 자동으로 채워준 것을 확인할 수 있습니다.
> 제목, 아티스트 이름을 기반으로 자동으로 채워지기 때문에 정확한 입력이 필요합니다.
